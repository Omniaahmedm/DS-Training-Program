## ðŸ“š Curriculum Roadmap

### Phase 1: The Foundation (8 Weeks)
Focuses on data literacy, cleaning, and supervised machine learning.
* **Outcome:** Students can clean messy data and build predictive regression/classification models.
* **Core Skills:** EDA, Feature Engineering, Regression, and Classification.

### Phase 2: Advanced Project Track (5 Weeks)
Focuses on industry-specific "Applied AI" and deployment.
1. **Real Estate Prediction:** Regression + FastAPI Deployment.
2. **Fraud Detection:** Anomaly detection + Financial ROI analysis.
3. **Insurance OCR:** Computer Vision + Streamlit UI.
4. **Cyber Security:** Autoencoders for threat detection.
5. **Introduction to LLMs & Generative AI:** Building RAG-based assistants.
